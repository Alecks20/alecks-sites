🌱 service owned and operated byalecks.dev

Welcome to my cozy little static web hosting service, I'm offering free static sites with unmetered bandwidth, a free alecks.site domain and integration with any git service.

I believe in bringing back creativity on the web and heavily support places like 32bit.cafe, neocities and others and intend for this to be a little digital garden of indie websites.

To get a website, either email me (me@alecks.dev) or dm me on discord (alecks20) and include your desired site name, email and git repo (plus api key if its private).

🍣 features and limits

network - bandwidth is shared and unmetered, however service access may be terminated (with warning) if abused.

automated updates - your git repo branch is pulled every 3 minutes to ensure site updates are instant like magic.

custom domains - I support custom domains for free, reach out to me and create a cname to alecks.site then I'll set it up.
